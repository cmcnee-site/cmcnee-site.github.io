How to Set Toon Render Settings in Poser

By Colin McNee


Step 1: Select Render Settings

Open the Render Settings Dialog Box (Command-Y)

Select the Preview tab.

Set the Texture Display to 4096, this will increase the level of detail in your render, but may tax a less capable system. You will need to experiment a little to balance the detail and performance.

Step 2: Adjust the Light/Shadow Settings

In the Pose Room, select the Main Light (you should use no more than two lights for this render).

In the Properties tab, select the Infinite Light button.

Adjust the Preview Shadow Map Size to 4096, this will increase the level of detail in the shadows of your render, this may also tax your system but the shadow map will vastly improve the look of your final render.

Step 3: Set Comic Book Preview Options

Select Comic Book Preview Options at the bottom left of the preview window.

Check Geometric Edge Lines and adjust the dial or numerical value to your liking (also check the Weld button to ensure the lines behave as expected.)

Select either Black and White or Color, depending on your intended use for the render. Note that Black and White will allow you to color in another application such as Photoshop or Affinity Designer. This is especcially useful in conjunction with exported G-Buffers in the Pro version of Poser 11.

Step 4: Render

Select Render (or Command-R) from the Render Menu.

When the render is complete, name and save the image

Note that options include .jpegs of various resolutions or .psd files (the G-Buffer output exported as layered files which can be exported to other graphics programs for further editing)
